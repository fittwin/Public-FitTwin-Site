Description
-----------
This module provides shopping cart block that just displays: a 16x16 shopping
cart icon, number of items in the cart, and the cart total, all linked to the
cart page.


Installation
------------
To install, copy the uc_microcart directory and all its contents to your
modules directory.

Configuration
-------------
To enable this module, visit administer -> site building -> blocks -> list page
and enable the Microcart block.
